# AryaStay – Hotel Booking System

AryaStay is a modern **Hotel Booking Web Application** that allows users to search hotels, select check-in/check-out dates, and book rooms easily. The project is built using **HTML, CSS, JavaScript, Node.js, Express, and MongoDB**.

---

## 🌐 Features

* 🏨 Hotel search by destination
* 📅 Check-in and check-out date selection
* 👥 Guest count selection
* 🔍 Search functionality with results page
* 📱 Responsive UI design
* 🔐 Admin login system
* 📊 Admin dashboard for viewing bookings
* 🗂 MongoDB database for storing booking data
* 🧭 Navbar navigation (Home, Hotels, Experience, About)

---

## ⚙ Installation

1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/aryastay-hotel-booking.git
```

2️⃣ Go to the project folder

```bash
cd aryastay-hotel-booking
```

3️⃣ Install dependencies

```bash
npm install
```

4️⃣ Start MongoDB (make sure MongoDB is running)

```
mongodb://127.0.0.1:27017/hotelDB
```

5️⃣ Start the server

```bash
node server.js
```

6️⃣ Open browser

```
http://localhost:3000
```
